# Ria — Python Syllabus 2026

## 10-Week Project-Based Plan (April – June 2026)

Each week = one lesson. Every lesson builds something real that you keep.

---

### Project 1: Personal Website with Flask (Weeks 1–4)

You've already started this! We'll keep going and turn it into a proper multi-page site you can show people.

| Week | Focus | What You'll Build / Do |
|------|-------|------------------------|
| **1** | **Pages & templates** | Add new pages (hobbies, favourites, contact). Practice HTML inside Jinja templates. Use `{% block %}`, headings, paragraphs, images, and links. |
| **2** | **Python → HTML with variables & loops** | Pass Python lists and dictionaries into your templates. Build a "My Favourite Things" page that loops over data and displays it in a table or card layout. |
| **3** | **Forms & user input** | Add a guestbook or contact form using Flask's `request` object. Learn `GET` vs `POST`, reading form data, and showing it back on the page. |
| **4** | **Styling & deployment** | Add CSS to make the site look polished (colours, fonts, layout). Optionally deploy to a free host (PythonAnywhere or Render) so you can share the link. |

**By the end of Week 4** you'll have a live personal website you built yourself.

---

### Project 2: Quiz Game — Terminal App (Weeks 5–6)

A command-line quiz game that tests the player on a topic you choose.

| Week | Focus | What You'll Build / Do |
|------|-------|------------------------|
| **5** | **Lists, dicts & functions** | Store questions + answers in a data structure. Write functions to ask a question, check the answer, and keep score. Play the game in the terminal. |
| **6** | **Files & high scores** | Save high scores to a text/JSON file. Load them when the game starts. Add difficulty levels or categories. |

**Skills gained:** functions, data structures, file I/O, string formatting, control flow.

---

### Project 3: Data Explorer — Analysing Real Data (Weeks 7–8)

Pick a dataset that interests you (music charts, football stats, weather, movies) and explore it with Python.

| Week | Focus | What You'll Build / Do |
|------|-------|------------------------|
| **7** | **Reading & filtering data** | Load a CSV file. Use loops and conditions to answer questions about the data (e.g. "Which artist has the most #1 hits?"). |
| **8** | **Visualisation** | Use `matplotlib` to create bar charts, line graphs, or pie charts from your findings. Save the charts as images. |

**Skills gained:** working with files/CSV, data processing, using external libraries, basic plotting.

---

### Project 4: Chatbot or Story Generator (Weeks 9–10)

Build an interactive program that responds to the user — either a chatbot or a choose-your-own-adventure story.

| Week | Focus | What You'll Build / Do |
|------|-------|------------------------|
| **9** | **String manipulation & logic** | Build the conversation/story engine. Use `if`/`elif` trees or dictionaries to map user input to responses. |
| **10** | **Polish & present** | Add randomness, personality, or branching paths. Package the project neatly. Demo it! |

**Skills gained:** string methods, dictionaries, random module, program design, debugging.

---

## Summary of Skills Covered (Weeks 1–10)

- Variables, types, f-strings
- Lists, dictionaries, tuples
- Loops (`for`, `while`) and conditionals
- Functions and parameters
- File I/O (text and JSON)
- HTML/CSS basics
- Flask web framework (routes, templates, forms)
- External libraries (`matplotlib`)
- Reading CSV data
- Program design and debugging
